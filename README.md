# TicTacToe

A simple TicTacToe game application built with Flutter.

Includes two modes:
  1. SinglePlayer Mode
 
       A user playes against an AI which always tries to win. The Ai logic is implemented using the MiniMax Algorithm
       
       
  2. MultiPlayer Mode
  
       You can play with your friends!
